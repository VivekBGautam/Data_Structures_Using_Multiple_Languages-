# Assignment_49

Here contains multiple **C programs** that perform various operations on a **Singly Linear Linked List**.  
These programs focus on **mathematical operations, divisibility checks, and value-based counting**, helping to enhance linked list data processing skills.

---

## Programs Overview

- **program49_1.c** – Finds the **difference between the maximum and minimum elements** in the linked list.  
  **Function Prototype:** `int Difference(PNODE Head);`

- **program49_2.c** – Checks whether **all elements are positive** in the linked list.  
  **Function Prototype:** `bool CheckAllPositive(PNODE Head);`

- **program49_3.c** – Displays elements that are **divisible by 3**.  
  **Function Prototype:** `void DisplayDivByThree(PNODE Head);`

- **program49_4.c** – Counts the number of elements **divisible by 5**.  
  **Function Prototype:** `int CountDivByFive(PNODE Head);`

- **program49_5.c** – Counts the number of **two-digit numbers** in the linked list.  
  **Function Prototype:** `int CountTwoDigit(PNODE Head);`
