## Assignment_45

Here contains multiple **C programs** that perform various operations on a **Singly Linear Linked List**.  
These programs focus on **filtering elements, searching positions, and counting based on conditions**, helping to understand traversal, conditional logic, and data handling in linked lists.

---

## Programs Overview

- **program45_1.c** – Displays **only the odd elements** of the linked list.  
  **Function Prototype:** `void DisplayOdd(PNODE Head);`

- **program45_2.c** – Returns the **position of the first occurrence** of a given element.  
  **Function Prototype:** `int FirstOccur(PNODE Head, int No);`

- **program45_3.c** – Returns the **position of the last occurrence** of a given element.  
  **Function Prototype:** `int LastOccur(PNODE Head, int No);`

- **program45_4.c** – Counts the number of elements **greater than a given value**.  
  **Function Prototype:** `int CountGreater(PNODE Head, int X);`

- **program45_5.c** – Counts the number of elements **less than a given value**.  
  **Function Prototype:** `int CountLess(PNODE Head, int X);`
