This repository contains multiple C programs demonstrating various operations on Singly Linked Lists. Each program focuses on a specific functionality, helping learners understand dynamic memory allocation, pointer manipulation, and standard linked list algorithms.

## Programs

## 1. Replace Negative Numbers with Absolute Value

### Filename: program50_1.c

Function Prototype: void MakeAbsolute(PPNODE Head);

Description: Traverses the linked list and replaces all negative numbers with their absolute values.

## 2. Display Elements Greater Than Average

### Filename: program50_2.c

Function Prototype: void DisplayGreaterThanAvg(PNODE Head);

Description: Calculates the average of all elements in the list and displays only those elements greater than the average.

## 3. Check Whether List is Sorted (Ascending Order)

### Filename: program50_3.c

Function Prototype: bool CheckSorted(PNODE Head);

Description: Checks whether the elements of the linked list are in ascending order. Returns true if sorted, otherwise false.

## 4. Display Alternate Nodes

### Filename: program50_4.c

Function Prototype: void DisplayAlternate(PNODE Head);

Description: Displays every alternate node in the linked list starting from the first node.

## 5. Count Nodes at Nth Position

### Filename: program50_5.c

Function Prototype: void Display(PNODE Head, int pos);

Description: Counts and displays the number of nodes present at a given Nth position in the linked list.
